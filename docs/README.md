## Краткий обзор архитектуры

```
┌─────────────┐    WebSocket     ┌─────────────┐
│  Frontend   │ ───────────────► │   Backend   │
│ (Streamlit) │                  │  (FastAPI)  │
│             │ ◄─────────────── │             │
└─────────────┘    JSON/b64      └─────────────┘
                                        │
                                        ▼
                                  ┌───────────┐
                                  │   Redis   │
                                  │(хранение) │
                                  └───────────┘
```

## Основные компоненты

- **Backend**: FastAPI + WebSocket, анализ видео через MediaPipe + PyTorch
- **Storage**: Redis для хранения комнат и клиентов
- **Frontend**: Streamlit для визуализации

## Версии зависимостей

См. `pyproject.toml` в директории `backend/`.
