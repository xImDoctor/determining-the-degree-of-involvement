# Уровни engagement и интерпретация диапазонов

---

## Классификация `level`

Реализация: [`EngagementCalculator._classify_level`](../../backend/app/services/video_processing/engagement_calculator.py#L317-L334).

`THRESHOLDS` ([engagement_calculator.py:101-105](../../backend/app/services/video_processing/engagement_calculator.py#L101-L105)):

| `level` | Порог `score` | Интерпретация |
|---------|---------------|---------------|
| `High` | ≥ 0.75 | Высокая вовлечённость |
| `Medium` | ≥ 0.50 | Средняя – вовлечённость присутствует, но не полностью сосредоточен |
| `Low` | ≥ 0.25 | Низкая – внимание рассеяно |
| `Very Low` | < 0.25 | Очень низкая или отсутствует |

> Проверка идёт сверху вниз – возвращается первая удовлетворяющая ветка.

## Интерпретация диапазонов

| Диапазон | `level` | Что обычно значит на практике |
|----------|---------|-------------------------------|
| 0.90–1.00 | `High` | Активная улыбка/удивление + прямой взгляд + нормальная активность глаз |
| 0.75–0.90 | `High` | Нейтральное выражение + внимательный взгляд, либо позитивная эмоция с лёгкой неуверенностью |
| 0.60–0.75 | `Medium` | Нейтрально или спокойная вовлечённость; короткие отвлечения |
| 0.50–0.60 | `Medium` | Состояние на грани – может быть и задумчивость, и читаемая усталость |
| 0.35–0.50 | `Low` | Заметные отвлечения, частые повороты головы, признаки усталости |
| 0.25–0.35 | `Low` | Комбинация негативной эмоции + отвлечение, либо длительный отворот от экрана |
| 0.00–0.25 | `Very Low` | Студент не в фокусе: голова сильно отвёрнута + глаза закрыты/усталые + негативная эмоция |

> Не следует делать выводы о вовлечённости по первым кадрам, когда сглаживание ещё неактивно, а тренд не обновляется. Вместо этого используйте `trend` и/или `score` за 10–30 секунд или более.

## Тренд как дополнительный сигнал

`level` и `trend` независимы – в UI/логике стоит их комбинировать:

| `level` | `trend` | Интерпретация |
|---------|---------|---------------|
| `High` | `rising` | Активно возрастает интерес |
| `High` | `stable` | Стабильная высокая вовлечённость |
| `High` | `falling` | Возможен старт потери фокуса |
| `Medium` | `rising` | Возврат внимания, положительная динамика |
| `Medium` | `falling` | Начало ухода в `Low` |
| `Low` | `stable` | Длительно низкая вовлечённость |
| `Low` | `rising` | Восстановление фокуса |
| `Very Low` | любое | Возможно, вовлечённость отсутствует |

## Связь `score` и `score_raw`

`score_raw` может быть значительно отличаться от `score` при резких событиях:

| Сценарий | `score_raw` | `score` | Комментарий |
|----------|-------------|---------|-------------|
| Улыбка на 1 кадр | `0.95` | `~0.72` | Сглаженный лаг – вариативный режим окна |
| Стабильная Alert+Happy 5+ сек | `~0.93` | `~0.93` | Система в стабильном режиме, окна совпадают |
| Зевок (глаза закрыты, 1 кадр) | `0.30` | `~0.65` | Сырое значение падает, сглаженное – ещё нет |

Для интерпретации пользователю **показывается `score`**, а `score_raw` служит только для отладки и анализа логов.

## Пороговые границы и их обоснование

Значения `0.75 / 0.50 / 0.25` подобраны под шкалу `[0, 1]` как равномерные квартили с небольшим смещением вверх для `High`. Такой выбор практически удобен и структурно согласован с несколькими академическими источниками.

### Согласованность с разметкой датасетов

DAiSEE (Gupta et al., arXiv:1609.01885) и EngageWild (Kaur et al., DICTA 2018, arXiv:1804.00858) используют ровно четыре уровня вовлечённости с аннотацией психологами: *Very Low / Low / High / Very High*. 

Четырёхклассовая структура подтверждена экспериментально: Whitehill et al. (*IEEE Transactions on Affective Computing*, 5(1), 2014) показали, что именно четыре уровня устойчиво различаются независимыми наблюдателями (κ > 0.6).

### Согласованность с EmotiW Engagement Challenge

В EmotiW Engagement Prediction sub-challenge (Dhall et al., *ICMI 2018*, DOI: 10.1145/3242969.3264993) целевая переменная задаётся как непрерывное значение в **[0, 1]** и квантуется на четыре уровня с точками 0 / ~0.33 / ~0.66 / 1.0. 

### Конечные пороги

Таким образом, **пороги `0.25` / `0.50` / `0.75`** – это границы между уровнями, округлённые до симметричных квартилей. Выбор согласован с формулировками EmotiW и DAiSEE.

Подробнее – в [literature.md](literature.md#пороги-классификации-вовлечённости-025-050-075).

Параметры `THRESHOLDS` заданы как class-attribute:

```python
# engagement_calculator.py:101-105
THRESHOLDS = {"high": 0.75, "medium": 0.50, "low": 0.25}
```

> Нет env-переменных; изменение только через код. При существенной смене весов компонентов или модели эмоций может потребоваться перекалибровка текущих параметров.

## Статистика сессии

Дополнительный метод [`get_statistics()`](../../backend/app/services/video_processing/engagement_calculator.py#L360-L379) возвращает aggregate по текущему окну `engagement_history`:

```python
{
    "mean": float,                  # среднее score за окно
    "std": float,                   # стандартное отклонение
    "min": float,
    "max": float,
    "total_frames": int,            # всего кадров обработано
    "current_window_size": int,     # фактический размер deque (до max 45)
}
```

Не используется в текущих WS-ответах, однако заложен для будущей аналитической логики. Полезен для пост-сессионной аналитики – например, расчёт среднего показателя вовлечённости за час занятия.
